D3 - Plan Mode, commando's en hooks

Leer geavanceerde functies van Claude Code: eerst plannen dan uitvoeren, handige commando's en automatische acties.

Na deze module kun je:

  • Plan Mode gebruiken voor complexe taken
  • De belangrijkste slash-commando's toepassen
  • Hooks instellen voor terugkerende acties
  • Commando's en Plan Mode combineren voor maximale controle
Geschatte tijd: 12 minuten
Module 1 van 4

Plan Mode

Standaard voert Claude Code direct uit wat je vraagt. Plan Mode verandert dat: Claude maakt eerst een gedetailleerd plan en wacht op jouw goedkeuring voordat hij begint.

Je schakelt Plan Mode in door voor je opdracht te typen: /plan [jouw opdracht]

Het Plan Mode-proces

/plan + opdracht

Jij beschrijft de taak

►

Plan

Claude beschrijft elke stap

►

Review

Jij keurt goed of past aan

►

Uitvoering

Claude voert het plan uit

Tip

Gebruik Plan Mode voor taken met veel stappen, taken op grote aantallen bestanden, of elke keer dat je niet zeker weet wat Claude gaat doen. Het kost iets meer tijd maar geeft je volledige controle.

Veelgebruikte commando's

Slash-commando's

CommandoWat het doet
/helpToont een overzicht van beschikbare commando's
/clearWist het huidige gesprek en begint schoon
/memoryToont wat Claude Code onthouden heeft
/costToont de API-kosten van de huidige sessie
/exit of Ctrl+CSluit Claude Code af
Tip

Typ /help als je even niet weet welke commando's beschikbaar zijn. Dit toont een overzicht met uitleg per commando.

Praktijksituatie

Module 3 van 4

Hooks: automatische acties

Hooks zijn automatische acties die Claude Code uitvoert op vaste momenten, zonder dat je er elke keer om hoeft te vragen. Je stelt een Hook in via je CLAUDE.md of door Claude Code te instrueren.

Klik op een type hook voor een voorbeeld

Bij sessiestart
"Aan het begin van elke sessie: toon een overzicht van alle bestanden in de map Nieuw die vandaag zijn toegevoegd."
Klik voor voorbeeld
Na een opdracht
"Na het aanmaken van een samenvatting: voeg de bestandsnaam en datum toe aan het logbestand sessie-log.txt."
Klik voor voorbeeld
Bij een specifieke actie
"Elke keer dat je een bestand overschrijft: maak eerst een back-up in de map Archief."
Klik voor voorbeeld
Let op

Complexe Hooks kunnen onverwacht gedrag veroorzaken. Begin eenvoudig met een of twee Hooks en bouw uit naarmate je meer ervaring krijgt.

Module 4 van 4

Combineren voor maximale controle

Je kunt commando's en Plan Mode combineren. Bijvoorbeeld: "/plan Gebruik de Hooks uit CLAUDE.md en verwerk de bestanden in de map Inbox." Claude maakt dan een plan inclusief alle automatische acties, zodat je precies weet wat er gaat gebeuren.

Samenvatting: wanneer gebruik je wat?

FunctieWanneer gebruiken
Plan ModeBij complexe taken of als je niet zeker bent wat Claude gaat doen
/clearAls het gesprek onoverzichtelijk is geworden
/costOm je API-kosten in de gaten te houden
HooksVoor terugkerende acties die je niet telkens wilt herhalen
Tip

Begin met Plan Mode bij je eerste grote taken. Voeg Hooks toe als je patronen herkent die je steeds herhaalt. Gebruik /cost om bewust te blijven van je API-gebruik.
