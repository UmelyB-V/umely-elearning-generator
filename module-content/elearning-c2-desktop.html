  • De desktop-app installeren en gebruiken

Module 1 van 4

De desktop-app installeren

Aan de slag

1

Download de app

Ga naar claude.ai/download en download de versie voor Mac of Windows.

2

Installeer en log in

Gebruik hetzelfde account als de web-app. Je hoeft geen nieuw account aan te maken.

3

Kies je functie

Je kunt kiezen tussen Chat, Cowork en Code. Voor de meeste gebruikers is Chat het startpunt.

Klik op een functie voor de uitleg

Chat
Hetzelfde gespreksvenster als de web-app. Typen, vragen stellen, bestanden uploaden. Maar sneller te openen en kan bestanden op je computer lezen.
Klik voor uitleg
Cowork
Claude voert taken uit op je computer: bestanden lezen, mappen aanmaken, documenten bewerken. Kan ook het web op via de browserfunctie en applicaties bedienen via computer use. Vraagt altijd bevestiging voordat hij iets doet.
Klik voor uitleg
Code
Voor het schrijven, uitleggen en debuggen van code. Relevant voor bedrijven die tools of automatiseringen willen bouwen. Niet nodig als je niet programmeert.

Module 2 van 4

Cowork: Claude voert taken uit

Cowork gaat verder dan alleen antwoorden geven. Claude neemt stappen op je computer: bestanden lezen, mappen aanmaken, documenten bewerken. Dit is krachtig maar vereist aandacht.

Claude vraagt altijd om bevestiging voordat hij iets uitvoert wat niet ongedaan gemaakt kan worden. Lees altijd wat hij van plan is voordat je bevestigt.

Praktijksituatie

Je hebt een map met 20 PDF-rapporten en wilt van elk rapport een samenvatting. Met Cowork kun je dit automatiseren. Wat doet Claude?

Module 3 van 4

Beperkingen en de Code-functie

Let op

Cowork werkt het beste voor bestandsbeheer en documentverwerking. Met de computer use functie kan Claude programma's openen, websites bezoeken en applicaties bedienen. Standaard is dit uitgeschakeld. Je moet het expliciet inschakelen.

De Code-functie is specifiek voor mensen die code schrijven of willen leren. Claude kan code genereren, uitleggen, debuggen en verbeteren. Dit is relevant voor bedrijven die intern tools willen bouwen.

Voor niet-developers: je hoeft hier niets mee te doen. Maar weet dat het bestaat. Als je op een punt komt dat je wilt automatiseren, is dit waar je begint.

Wat kan Cowork wel en niet?

Module 4 van 4

Desktop-app vs web-app

Wanneer kies je welke?

SituatieBeste keuze
Je gebruikt Claude regelmatig en wilt hem snel openenDesktop-app
Je wilt bestanden op je computer laten verwerkenDesktop-app (Cowork)
Je wilt snel iets vragen zonder installatieWeb-app
Je werkt op een ander apparaatWeb-app
Je werkt samen met collega's via TeamsWeb-app
Tip

Je hoeft niet te kiezen. Veel gebruikers hebben zowel de desktop-app als de web-app. Je account en gesprekken zijn op beide beschikbaar.
