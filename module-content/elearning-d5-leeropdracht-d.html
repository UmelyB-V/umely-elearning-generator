Leeropdracht D — Claude Code opzetten via VS Code

Je doorloopt alle stappen van D1 tot D4. Aan het einde heb je Claude Code geinstalleerd via de VS Code-extensie, een CLAUDE.md geschreven en Plan Mode gebruikt.

Vul je naam in om verder te gaan.

Hoe deze leeropdracht werkt

1
Open VS Code naast dit scherm — je werkt vandaag met de Claude Code-extensie.
2
Elke fase heeft concrete stappen die je in VS Code uitvoert.
3
Kom terug, vul je verificatieveld in en ga door.
Geschatte tijd: 30-45 minuten

Na deze leeropdracht heb je:

  • Bepaald voor welke taak Claude Code het meest waardevol is in jouw werk
  • De Claude Code-extensie geinstalleerd in VS Code
  • Je eerste opdracht gegeven en een CLAUDE.md geschreven
  • Plan Mode gebruikt om een taak te plannen voordat Claude hem uitvoert

In D1 heb je geleerd wanneer je meer nodig hebt dan de web-app. Claude Code is geschikt als je met bestanden werkt, meerdere stappen achter elkaar wil uitvoeren, of herhalende taken wil automatiseren. In deze fase bepaal je concreet welke taak jij met Claude Code wil doen.

Voorbeelden van Claude Code-taken

Documenten hernoemen
Hernoem alle Word-bestanden in een map zodat ze beginnen met de datum in het formaat JJJJ-MM-DD.
Klik voor voorbeeld
Mappen aanmaken
Maak een mapstructuur aan op je bureaublad: Klanten > [klantnaam] > Offertes, Facturen, Correspondentie.
Klik voor voorbeeld
Bestanden doorzoeken
Zoek alle Excel-bestanden in een map die de afgelopen 30 dagen niet zijn aangepast.
Klik voor voorbeeld
Minimaal 20 tekens. Wees zo concreet mogelijk — welke bestanden, welke map?
Fase 2 van 4

Installeer de extensie en geef je eerste opdracht

✓
2
3
4

De VS Code-extensie is de eenvoudigste manier om met Claude Code te starten — geen terminal, geen Node.js installatie. Je installeert de extensie, logt in met je Anthropic-account en bent klaar.

Installatiestappen

1
Open VS Code.
2
Klik op het Extensions-icoon in de linkerzijbalk (vier vierkantjes) of gebruik Ctrl+Shift+X.
3
Zoek op 'Claude Code'.
4
Klik op Install.
5
Log in met je Anthropic-account via de instructies die verschijnen.

Je eerste opdracht

Nu Claude Code actief is, geef je hem de taak die je in fase 1 hebt beschreven.

Tip

Beschrijf de taak zo concreet mogelijk: welke map, welk patroon, welk resultaat. Claude vraagt om bevestiging bij ingrijpende acties — lees altijd wat hij van plan is.

Minimaal 20 tekens.
Fase 3 van 4

Schrijf een CLAUDE.md

✓
✓
3
4

Claude Code begint elke sessie blanco — hij heeft geen geheugen van vorige gesprekken. Een CLAUDE.md is het projectgeheugen: een bestand in de hoofdmap van je project dat beschrijft wat Claude altijd moet weten. Je schrijft het eenmalig en hij gebruikt het automatisch bij elke start.

Kopieer en vul deze CLAUDE.md-template in
# [Naam van je project] ## Doel [Beschrijf in één zin wat dit project is of wat je ermee doet] ## Taal Schrijf altijd in het Nederlands. ## Vaste regels - [Regel 1 — bijv. sla bestanden altijd op in de map /output] - [Regel 2 — bijv. vraag om bevestiging voordat je iets verwijdert]
1
Kopieer de template.
2
Open een nieuw bestand in VS Code.
3
Vul de template in voor het project of de map waarmee je werkt.
4
Sla het op als CLAUDE.md in de hoofdmap van je project.
Minimaal 20 tekens.
Fase 4 van 4

Gebruik Plan Mode

✓
✓
✓
4

Plan Mode laat Claude eerst nadenken voor hij iets uitvoert. Je geeft de opdracht, Claude schrijft een stap-voor-stap plan en wacht op jouw goedkeuring. Zo houd je de controle — zeker bij taken die meerdere bestanden aanraken.

Geef deze opdracht in Claude Code (met /plan vooraf)
/plan Maak een overzicht van alle bestanden in de huidige map met per bestand een korte beschrijving van wat het bevat of waarvoor het gebruikt wordt.
1
Kopieer de opdracht.
2
Plak hem in Claude Code in VS Code.
3
Lees het plan dat Claude schrijft.
4
Pas één stap aan als je het niet eens bent, of keur het goed.
Tip

Gebruik /plan altijd bij taken die bestanden aanpassen, verplaatsen of verwijderen. Bij leestaken (overzicht maken, samenvatten) is Plan Mode optioneel.

Minimaal 20 tekens.
