Module 1 van 4

Waarom en wat heb je nodig?

Een eenvoudige website bouwen kostte vroeger duizenden euro's en een developer. Met Claude Code kun je een nette, functionele website maken in een middag, ook als je nog nooit een regel code hebt geschreven.

Dit is niet bedoeld voor complexe webapplicaties of webshops met betalingen. Het is bedoeld voor een eenvoudige aanwezigheid online: wie ben je, wat doe je, hoe kun je contact opnemen.

Wat heb je nodig?

1

Claude Code

Geinstalleerd en werkend (zie module D1).

2

Een webbrowser

Om het resultaat te bekijken. Chrome, Firefox of Edge.

3

Optioneel: hostingaccount

Netlify of GitHub Pages (gratis) om de website online te zetten.

Module 3 van 4

Online zetten

Website live zetten via Netlify

1

Maak een Netlify-account

Ga naar netlify.com en maak een gratis account aan (gratis tier beschikbaar op moment van schrijven — controleer netlify.com voor de actuele voorwaarden).

2

Upload je website

Sleep je websitemap naar het Netlify-dashboard.

3

Website is live

Netlify geeft je een URL. Je website is direct bereikbaar.

Tip

Voor een eigen domeinnaam (www.jouwbedrijf.nl) koop je een domein bij een registrar zoals TransIP of Antagonist. De kosten zijn laag — check de actuele tarieven op de website van de registrar.

Module 4 van 4

Beperkingen en onderhoud

Wat kan Claude wel en waar heb je meer kennis nodig?

Kan zelfstandigKan, maar vraagt meer begeleiding
HTML en CSS genererenBetalingssysteem integreren (Stripe, Mollie)
Teksten en kleuren aanpassenInlogsysteem bouwen (authenticatie, sessies)
Mobiele weergave verbeterenDatabase koppelen (vereist technische kennis)
Secties toevoegen/verwijderenAfbeeldingen genereren of uploaden

Claude Code kan eenvoudige webapplicaties bouwen, inclusief inlogsystemen en databasekoppelingen. Voor complexe systemen met betalingen en gevoelige data is meer technische kennis en begeleiding nodig.

Let op

Een website die Claude heeft gebouwd, moet je zelf onderhouden. Als je teksten wilt aanpassen, doe je dat via Claude Code of door de HTML-bestanden direct te bewerken. Claude kan geen afbeeldingen genereren, maar je voegt eigen afbeeldingen toe door ze in de juiste map te plaatsen.

Tip

Als je website complex wordt, overweeg dan een CMS zoals WordPress. Maar voor een eenvoudige bedrijfswebsite is een door Claude gebouwd HTML-bestand meer dan voldoende.
