Module 1 van 4

Wat zijn Plugins?

Plugins bundelen meerdere functies samen voor een specifieke afdeling of werksituatie. Waar een Connector een tool koppelt (bijv. Gmail), bundelt een Plugin meerdere Connectors, instructies en commando's voor een complete werkflow.

Connector vs Plugin

ConnectorPlugin
Koppelt een toolBundelt meerdere tools + instructies
BouwsteenComplete werkoplossing
Voor losse takenVoor terugkerende werkflows
Bijv. Gmail koppelenBijv. complete HR-workflow

Claude bouwt de Plugin

Via Plugin Create stelt Claude je vragen over wat je wilt en genereert daarna de Plugin voor je. Je hoeft geen technische kennis te hebben. Wel moet je goed weten wat je wilt: welke werkflow, welke tools, welke instructies.

Tip

Begin niet met bouwen. Begin met nadenken. Welke taken doe je elke week? Welke tools gebruik je daarbij? Welke instructies herhaal je steeds? Die antwoorden vormen je Plugin.

Module 4 van 4

Wanneer Plugin vs Connector?

Keuzewijzer

SituatieKeuze
Je wilt Claude koppelen aan een tool voor losse takenConnector
Je hebt een terugkerende werkflow met meerdere toolsPlugin
Je wilt snel aan de slag zonder setupConnector
Je wilt vaste instructies en commando's voor een afdelingPlugin
Let op

Plugins vereisen een goede setup. Een slecht geconfigureerde Plugin geeft inconsistente resultaten. Test altijd grondig voordat je een Plugin breed uitrolt.
